# Blog App

A full-stack Blog Application where users can create, read, update, and delete blog posts.

## 🚀 Features

- User Authentication
- Create Blog Posts
- Edit and Delete Posts
- Responsive UI
- REST API Integration
- MongoDB Database
- JWT Authentication

## 🛠 Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## 📂 Project Structure

```bash
BlogApp/
│── frontend/
│── backend/
│── README.md
```

## ⚙️ Installation

### Clone the repository

```bash
git clone <your-repository-link>
```

### Install backend dependencies

```bash
cd backend
npm install
```

### Install frontend dependencies

```bash
cd frontend
npm install
```

## ▶️ Run the Application

### Start backend

```bash
npm start
```

### Start frontend

```bash
npm run dev
```

## 🌐 Environment Variables

Create a `.env` file in backend folder and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
